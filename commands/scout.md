# Scout

Run the **codebase-assessor** agent to analyze the current codebase:

1. Stack inventory (languages, frameworks, deployment target)
2. Schema shape (model files, column counts, relationship graph)
3. Test health (coverage, test types, gaps)
4. Churn analysis (most-modified files, fix ratio)
5. Deployment constraints (platform limits, auth patterns)

Output a structured assessment with recommended next steps.
