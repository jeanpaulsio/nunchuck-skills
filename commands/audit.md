# Audit

Run the Ship Confident audit cadences. Specify frequency:

- `/audit daily` -- Review today's diff for duplication, coupling, naming, testability
- `/audit weekly` -- Cross-PR analysis for repeated patterns, hot files, growing files
- `/audit monthly` -- Full codebase shape review with ranked backlog output

Each cadence checks both **code quality** and **test health**.

See skill: `ship-confident` for the full checklist at each cadence.
